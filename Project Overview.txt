Project Overview:
This project focuses on analyzing banking transaction data to uncover fraud patterns and customer behavior using SQL.

Key Objectives:

1.Perform data cleaning and preprocessing
2.Analyze transaction trends and distributions
3.Detect fraudulent transactions using SQL-based rules
4.Identify high-risk customers and unusual transaction patterns

Key Features:

1.Fraud detection using balance validation and transaction thresholds
2.Customer segmentation based on transaction activity
3.time-based transaction analysis
4.Advanced SQL queries including window functions and subqueries

Tools Used:

SQL (MySQL/PostgreSQL)
